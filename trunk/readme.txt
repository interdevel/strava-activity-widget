== Description ==

A simple plugin to display a widget with your activity on Strava. This can be a list of your recent rides/runs, or a summary of your last week of riding/running.

== Installation ==

1. Unzip the zip file in your plugins directory (usually wp-content/plugins).
1. In WordPress admin go to 'Plugins' -> 'Installed Plugins' and activate the Strava Activity Widget plugin.
1. Go to the 'Appearance' -> 'Widgets' page, drag the Strava Activity widget into the sidebar of your choice. You will be able to enter a title for the widget there, choose the type of information to show and also the dimensions of the widget (in pixels).
1. Enjoy!
